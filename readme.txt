Teknix Admin UI offers branding, menu customization and widget selection options for the WordPress admin interface.


== Description ==

Teknix Admin UI is a lightweight plugin for customizing your WordPress interface for your clients.

**Features**

* Hide uneccessary links from navigation menu for all non-administrator users
* Replace WordPress logo with custom text or logo for branding purposes
* Remove default widgets on dashboard (Events/News and Welcome panel) and add custom widget with HTML support

== Installation ==

If you cannot install through WordPress plugin repository, then you can use the following instructions:

1. Download teknix-admin-ui.zip archive
2. Unzip the zip file and upload the directory to your `/wp-content/plugins/` directory
3. Go to `Plugins` menu in Wordpress and activate Teknix Admin UI
